# Implementation Notes

## Current status

The repository now contains a lightweight research prototype for quasicrystalline-inspired layouts. The code focuses on:

- generating non-periodic point sets
- estimating spacing-based coherence proxies
- estimating simplified crosstalk risk
- providing a small test suite for regression checks

## Next steps

1. Add more elaborate topologies such as Penrose and Ammann-Beenker tilings.
2. Integrate a quantum simulation layer using Qiskit or a simplified circuit model.
3. Create a results report with plots for coherence and crosstalk metrics.
4. Translate the research plan into a formal milestone tracker.
